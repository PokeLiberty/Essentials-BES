Module: Events

Defined in:
scripts/PField_Field.rb

Overview

This module stores events that can happen during the game. A procedure can subscribe to an event by adding itself to the event. It will then be called whenever the event occurs.

false, some other code had called $scene.createSpritesets to regenerate the
map scene without transferring the player elsewhere

.onStepTakenTransferPossible=(v) ⇒ Object

Fires whenever the player takes a step. The event handler may possibly move the player elsewhere. Parameters: e = Array that contains a single boolean value.

If an event handler moves the player to a new map, it should set this value

to true. Other event handlers should check this parameter's value.
