Module: EncounterModifier

Defined in:
scripts/PField_Field.rb

Overview

This module stores encounter-modifying events that can happen during the game. A procedure can subscribe to an event by adding itself to the event. It will then be called whenever the event occurs.

def self.trigger(encounter)
  for prc in @@procs
    encounter=prc.call(encounter)
  end
  return encounter
end
