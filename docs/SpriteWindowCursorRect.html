Overview

SpriteWindow is a class based on Window which emulates Window's functionality. This class is necessary in order to change the viewport of windows (with viewport=) and to make windows fade in and out (with tone=).

def empty
  needupdate=@x!=0 || @y!=0 || @width!=0 || @height!=0
  if needupdate
    @x=0
    @y=0
    @width=0
    @height=0
    @window.width=@window.width
  end
end
